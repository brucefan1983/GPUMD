Input files
===========

To run a :term:`NEP` construction  using the ``nep`` executable, one has to prepare three input files that respectively specificy the parameters of the :ref:`NEP model <nep_in>` (``nep.in``), the :ref:`training dataset <train_test_xyz>` (``train.xyz``), and the :ref:`test dataset <train_test_xyz>` (``test.xyz``).
